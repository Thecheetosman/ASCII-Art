TERMINAL MANUAL & KEYBINDS

            
            
[ INPUT & FILE HANDLING ]

            
- Drag and drop images or videos anywhere on screen to load them.

            
- Or use the file uploader at the top left of the workspace.

            
- USE WEBCAM opens your raw video stream directly inside the processor.


            
[ KEYBOARD HOTKEYS ]

            
- SPACEBAR : Play / Pause video playback.

            
- , or < : Step 1 frame backward (fine timeline scrubbing).

            
- . or > : Step 1 frame forward (fine timeline scrubbing).

            
(Keys are automatically disabled when you are actively typing inside input fields.)


            
[ FORMATS & EXPORTING ]

            
- COPY ``` / ~~~ : Copies markdown block formats. Tildes prevent backtick collision.

            
- COPY ANSI : Copies 24-bit terminal RGB escape sequences to clipboard.

            
- EXPORT SEQUENTIAL FRAMES : Step-renders video directly. Select local directory folder for seamless writes (Chrome/Edge only) or run sequential downloads.

            
- RECORD .WEBM : Captures frames real-time and mixes the crushed 4-bit audio directly into a synchronized WebM video file.

            
- RECORD COLOR : Toggles color-pencil canvas painting on or off for the WebM recording pipeline.

            
- DL .HTML : If media is static, downloads current frame. If video is loaded, prompts to either export static frame or compile selected Trim Range as an animated HTML player.


            
[ TIMELINE TIMING TRIM ]

            
- Restrict standard loop playback, sequential frame exports, animated HTML compilations, and WebM video recordings using the Trim In and Trim Out numeric range markers.


            
[ MODULAR AUDIO REACTIVITY ]

            
- Enable Audio Reactive controls to let volume RMS scale, glitch skew, or flash text brightness cumulatively with the audio output.
